# RecoverAura - Vehicle Repossession Management SaaS

A production-ready SaaS application for managing vehicle repossession operations.

## Features
- React 18 + Vite frontend
- Supabase PostgreSQL backend
- Real-time authentication
- Dashboard with analytics
- Vehicle management
- Case tracking
- Responsive design

## Quick Start

1. `npm install`
2. `cp .env.example .env.local`
3. Add your Supabase credentials to `.env.local`
4. Import `database/SUPABASE_SCHEMA.sql` to Supabase
5. `npm run dev`

Access: http://localhost:5173

## Tech Stack
- Frontend: React 18, Vite, Tailwind CSS
- Backend: Supabase (PostgreSQL)
- Deployment: Docker, Vercel
